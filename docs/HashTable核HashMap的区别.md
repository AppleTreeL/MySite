##### 话不多说，看这个表

| 区别     | HashMap                  | HashTable                                          |
| -------- | ------------------------ | -------------------------------------------------- |
| 推出版本 | JDK1.2                   | JDK1.0                                             |
| 性能     | 异步处理，性能更高       | 同步处理，性能低                                   |
| 安全性   | 非线程安全               | 线程安全                                           |
| null操作 | 允许一个null，只允许一个 | Key和Value都不能为空，否则出现NullPointerException |
